# `@sk/mock-api`

Shared **prototype fixtures** that mimic API/database responses for SK Enterprises.

- **`getWebAdminMockBootstrap()`** — shapes for web Redux `preloadedState`
- **`getMobileMockBootstrap()`** — shapes for mobile Redux `preloadedState`
- **`mockApiEmployeesResponse` / `mockApiDashboardSummary`** — examples aligned with `docs/10-API-CONTRACT-EXAMPLES.md`
- **`mockDelay()`** — optional latency for async loaders

Edit files under `src/fixtures/` to change demo data in both apps at once.
